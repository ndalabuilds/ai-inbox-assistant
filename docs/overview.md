# Overview

## Goal

Show how an AI-assisted inbox workflow can reduce manual email handling for a small business.

## Problem

Many businesses lose time sorting emails manually:
- deciding what is urgent
- routing enquiries
- drafting replies
- extracting lead/customer details
- updating spreadsheets or CRMs

## Proposed solution

An AI inbox assistant can:

- classify messages
- identify intent
- extract structured information
- draft replies
- trigger next actions

## Example input

Subject: Need a quote for website redesign

Body:
Hi, we run a small accounting firm and need a new site with booking forms and a client portal. Could you share pricing and timeline?

## Example output

Category: Sales lead  
Urgency: Medium  
Summary: Small accounting firm requesting website redesign quote  
Key details:
- Industry: Accounting
- Needs: Website redesign, booking forms, client portal
- Intent: Request for quote

Suggested reply:
Hi, thanks for reaching out. I’d be happy to help. Based on your description, I can put together an initial recommendation and estimated timeline...
